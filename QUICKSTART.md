# 🚀 Guía Rápida

## 1. Cargar Imagen
Haz clic en "Cargar Imagen" → Selecciona una imagen

## 2. Crear Zona
Modo Dibujar → Arrastra para crear zona

## 3. Añadir Contenido
Selecciona zona → "Editar Contenido"

## 4. Guardar
Ctrl+S → Se guarda automáticamente

## 5. Exportar
Botón "Exportar" → Descarga HTML

## Atajos
- Ctrl+S: Guardar
- Ctrl+Z: Deshacer
- Ctrl+Y: Rehacer
- Ctrl+D: Duplicar
- Supr: Eliminar

Ver README.md para más detalles.
